Skill Reference

29 self-contained skills are the plugin's entire surface area — the plugin is agentless, so every reviewer and researcher lives inside the skill that uses it. Seven core-loop skills double as slash commands — type /js-ce:plan or /js-ce:code-review to run them. The rest are invoked by name (skill: js-ce-<name>) or triggered automatically by Claude when relevant.

How to Use Skills

# Core-loop skills run as colon-namespaced slash commands
/js-ce:plan "Add user authentication"
/js-ce:code-review 123

# Other skills are referenced by their hyphenated name
skill: js-ce-debug
skill: js-ce-simplify-code

Commands are skills now

There is no separate "commands" list. The workflow commands you may remember -- /js-ce:plan, /js-ce:code-review, /js-ce:work, /js-ce:compound, and the rest -- were migrated to skills. They live under skills/<name>/SKILL.md and work identically as slash commands in Claude Code.

Core Loop (5)

The primary compounding-engineering loop. All five are invoked as colon-namespaced slash commands.

js-ce-ideate

slash command

Generate and evaluate grounded ideas before choosing one to develop. 

/js-ce:ideate

js-ce-brainstorm

slash command

Explore a vague or ambitious idea into a right-sized requirements document. 

/js-ce:brainstorm

js-ce-plan

slash command

Create structured plans for any multi-step task, with automatic confidence checking. 

/js-ce:plan

js-ce-work

slash command

Execute a plan or concrete work prompt end-to-end. 

/js-ce:work

js-ce-compound

slash command

Document a solved problem or durable vocabulary to compound team knowledge. 

/js-ce:compound

Review & Refinement (5)

Review code and documents, then refine what you shipped. /js-ce:code-review and /js-ce:compound-refresh run as slash commands; the rest are invoked by name.

js-ce-code-review

slash command

Structured code review with dynamically selected reviewer personas, confidence gating, and a dedup pipeline. 

/js-ce:code-review

js-ce-simplify-code

Simplify recently changed code for clarity, reuse, quality, and efficiency while preserving behavior. 

skill: js-ce-simplify-code

js-ce-doc-review

Review requirements, plans, or specs with role-specific persona lenses. 

skill: js-ce-doc-review

js-ce-optimize

Run metric-driven optimization loops with parallel experiments and LLM-as-judge scoring. 

skill: js-ce-optimize

js-ce-compound-refresh

slash command

Refresh stale or drifting learnings and decide whether to keep, update, replace, or archive them. 

/js-ce:compound-refresh

On-Demand (3)

Reach for these whenever you're stuck, need something explained, or want a verdict.

js-ce-debug

Diagnosis loop for bugs and failing behavior -- traces causal chains and implements test-first fixes. 

skill: js-ce-debug

js-ce-explain

Turn a concept, diff, or window of recent work into a dense, visual explainer. 

skill: js-ce-explain

js-ce-pov

Give a decisive, project-grounded verdict on an external technology, library, or pattern. 

skill: js-ce-pov

js-ce-product-pulse

Generate a time-windowed pulse report from configured signals. 

skill: js-ce-product-pulse

js-ce-sweep

Sweep configured feedback sources (Slack, GitHub Issues) for new items and emit an lfg-ready plan. 

skill: js-ce-sweep
