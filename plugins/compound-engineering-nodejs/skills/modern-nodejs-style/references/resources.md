# Modern Node.js Resources

## Influential Engineers

### TJ Holowaychuk
Creator of Express, Koa, Mocha, Commander, and 500+ npm packages.

- **GitHub**: https://github.com/tj
- **Express.js**: https://github.com/expressjs/express
- **Koa**: https://github.com/koajs/koa
- **Commander**: https://github.com/tj/commander.js

### Matteo Collina
Node.js TSC member, creator of Fastify and Pino.

- **GitHub**: https://github.com/mcollina
- **Fastify**: https://github.com/fastify/fastify
- **Pino**: https://github.com/pinojs/pino
- **Platformatic**: https://platformatic.dev/

### Yusuke Wada
Creator of Hono, the ultrafast web framework for the edge.

- **GitHub**: https://github.com/yusukebe
- **Hono**: https://github.com/honojs/hono
- **Hono Docs**: https://hono.dev/

Most prolific npm author with 1000+ packages.

- **GitHub**: https://github.com/sindresorhus
